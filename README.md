# Pasta Cosi

Immersive restaurant website concept and implementation for Pasta Cosi.

## Stack
- Next.js 15 App Router
- TypeScript strict
- Tailwind CSS v4
- Framer Motion
- GSAP
- Lucide React

## Run
```bash
npm install
npm run dev
```

## Notes
- Hero and gallery use premium placeholders where final restaurant assets are missing.
- Replace the current image and gallery placeholders in `app/page.tsx` and `lib/site-data.ts`.
- Reservation form is presentational and ready to connect to ResDiary, SevenRooms, Formspree, or a custom endpoint.
