# Pasta Cosi - Project Brief

## Overview
Avant-garde restaurant website. Single-page, immersive, continuous scroll experience. **All content in French.**

## Core Requirements

### Design Philosophy
- **NOT** simple black/white backgrounds
- Rich, detailed textures: veined blanc crème marble, warm architectural concrete, brushed brass
- CSS blend modes + lighting shaders responsive to user interaction
- Fluid, physics-based scroll triggers (GSAP or Framer Motion)
- Elements drift into place elegantly
- Text/imagery integrated into background layers (etched into marble effect)

### Technical Standards
- Asynchronous loading architecture (zero friction/jumps)
- Sub-second paint times on all devices
- High-end aesthetic + perfect accessibility
- Lighthouse 100/100/100/100 target
- No menus or pricing lists on main experience

### Content
**Restaurant Name:** Pasta Cosi

**All text in French** - sensory storytelling focused on atmosphere

### Visual Assets
20 food/interior images provided (see attachments in shared workspace)

## Deliverables
1. Single-page website (Next.js/Remix/SvelteKit - your choice)
2. Immersive scroll experience with physics-based animations
3. Rich textured backgrounds (marble/concrete/brass)
4. Optimized images (WebP/AVIF, lazy-loaded)
5. Full accessibility (WCAG AA minimum)
6. Production-ready code

## Success Criteria
- Feels like a digital art piece, not just a website
- Users should feel the restaurant's atmosphere through the screen
- Perfect technical performance
- "Someone cared about this" signature quality
