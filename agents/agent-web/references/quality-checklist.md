# AgentWeb Quality Checklist

Use this checklist before delivering web-related output.

## Visual

- The direction feels intentional and premium.
- The layout is not a generic template clone.
- Typography, spacing, and alignment are consistent.
- The hero section creates immediate impact.
- Sections have a clear visual rhythm.

## UX

- The value proposition is understandable quickly.
- Calls to action are visible and well placed.
- Content hierarchy is easy to scan.
- Mobile behavior is considered, not implied.
- Navigation and flow are simple and clear.

## Technical

- Code is readable and maintainable.
- Responsive behavior is defined.
- Semantic HTML and accessibility basics are respected.
- Obvious performance issues are avoided.
- Interactions/animations do not create unnecessary heaviness.

## Business

- The page supports the intended goal.
- Trust elements are present when relevant.
- The structure supports conversion, not just aesthetics.
- Recommendations highlight the highest-leverage improvements first.
